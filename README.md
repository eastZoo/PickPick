# 🎞모두를 위한 유튜브 공유사이트
- 프로젝트 명칭: PickPick
- 목적: 연구회 사람들과 영상을 공유하여 재미있는 순간을 함께 나누기 위해서
- 주요 기능: 유튜브를 링크를 통한 공유 영상 추가
- 개발 인원: Front-End 1명, Back-End 2명
- 개발 기간: 2022.09 ~

# 기술 스택
- 협업 툴: Notion, Slack
- 형상관리 툴: Github

## Front
- 개발 언어: JavaScript
- 개발 환경: React

## Back
- 개발 언어: JAVA 11
- 개발 환경: SpringBoot 2.7.2, gradle 7.5, JPA, Oauth 2.0
- 데이터베이스: PostgreSQL
